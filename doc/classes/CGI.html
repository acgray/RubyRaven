Methods

Public Instance methods

redirect( where )

CGI doesn’t have a redirect method. Strange eh? Sets the 302 Moved header to point the url specified in where.

timeforRFC3339( rfc3339 )

Perhaps not a great place for this, but rfc3339 is an internet standard .… Takes a string with a time encoded according to rfc3339 and returns a Time object.
