Please see Raven for use instructions. You should also have a copy of ravenexample.rb along with this file. This gives a basic example of how to use.

Uses the standard library that comes with Ruby 1.8, may be backportable to Ruby 1.6 but I haven’t tested it.

Needs the host computer to have OpenSSL correctly installed (done by default on unix and mac?)
